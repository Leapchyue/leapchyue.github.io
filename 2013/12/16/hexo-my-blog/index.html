昨天折腾了不少时间，终于搭好博客了。

配置环境

环境是 windows 7

安装 git 和配置 github

按照github上面的指南安装git和生成ssh-key直接做就好了。

安装 Node.js 和 Hexo

安装 Node.js ，到Node.js官网下载直接安装就好。
