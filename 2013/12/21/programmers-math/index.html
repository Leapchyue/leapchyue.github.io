读《程序员的数学》

昨晚睡不着，只能起来看书，没得选床头只有一本《程序员的数学》，就只能看这本了。结果不一会就看完了，这本算是相当科普的书咯。话说高中的应该看完也是毫无鸭梨的。而至于什么是“程序员的数学”，我觉得这本书还谈不上，这本仅仅是入门基础而已，给人启发而已。

书的内容讲得很有趣比较吸引人，比较好懂，有不少以前没有想过的，这本书值得瞄下。但书上大部分的内容都不用怎么思考的，大部分都是以前见过的内容或者曾经思考过的，基本不到两个小时就可以看完。昨晚也就大概2个小时看完了。不过也有不少以前没有注意到的。

笔记

  • 0的作用：占位，统一标准，简化规则
  • 蕴涵——若A则B
A B $ A \Rightarrow B $

1 1 1
1 0 0 当A为真时，当且仅当B为假时，$ A \Rightarrow B $ 才为假
0 1 1
0 0 1
  • 由 A 和 B构成的所有运算的真值表
A 0 0 1 1
B 0 1 0 1
——- ——- ——- ——- ——- ————————————
0 0 0 0 0 恒为0
1 0 0 0 1 $ A \wedge B $
2 0 0 1 0 $ A \wedge (\neg B) $
3 0 0 1 1 $ A $
4 0 1 0 0 $ (\neg A) \wedge B $
5 0 1 0 1 $ B $
6 0 1 1 0 $ \neg(A = B) $ 即 $ A \oplus B $
7 0 1 1 1 $ A \vee B $
8 1 0 0 0 $ \neg(A \vee B) $
9 1 0 0 1 $ A = B $
10 1 0 1 0 $ \neg B $
11 1 0 1 1 $ A \vee (\neg B) $
12 1 1 0 0 $ \neg A $
13 1 1 0 1 $ (\neg A) \vee B $ 即 $ A \Rightarrow B $
14 1 1 1 0 $ \neg (A \vee B) $
15 1 1 1 1 恒为1

  • 余数
    算多少天后是星期几，包括$ x^y $ 天这种形式，注意有个循环节的概念，即是取余到一定程序后会循环，就如同循环小数那样有个循环节。

  • 奇偶校验
    在余数的内容中，有著名的七桥问题，还有下面这两个问题。

  • 找人问题，已知初始时人在哪，每天随机移动到一次，问多少天后在哪个位置的概率。
  • 有一个下面左边的房间，问能否用右边的席子铺满房间。

  • 置换、排列和组合
    这部分内容比较简单。
